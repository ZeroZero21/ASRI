## v1 - 2025-05-24

### Added
- Initial public release of the ASRI preprint repository.
- Added the preprint materials for the ASRI project.
- Added repository citation metadata through `CITATION.cff`.
- Added a custom license clarifying permitted use, citation expectations, and restrictions against plagiarism, false authorship claims, or misappropriation.
- Added basic repository metadata for academic reference and sharing.
- Enabled GitHub’s **Cite this repository** feature through citation configuration.
