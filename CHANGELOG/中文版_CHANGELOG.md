## v1 - 2025-05-24

### 新增
- ASRI 预印本仓库的首次公开发布。
- 添加 ASRI 项目的预印本相关材料。
- 通过 `CITATION.cff` 添加仓库引用元数据。
- 添加自定义许可证，明确允许使用范围、引用要求，以及对抄袭、冒名和盗用行为的限制。
- 添加用于学术引用与公开分享的基础仓库元信息。
- 通过引用配置启用 GitHub 的 **Cite this repository** 功能。

### 说明
- 此版本为 ASRI 仓库的首个公开预印本版本。
- 本仓库主要用于学术分享、阅读、讨论与引用。
- 后续版本可能包含修订、澄清、翻译扩展、补充材料或结构调整。
